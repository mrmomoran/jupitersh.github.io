急性移植物抗宿主反应(GVHD)分级

分级 皮肤（皮疹） 肝脏（胆红素） 胃肠道（腹泻）
0 无皮疹 <34μmol/L（<2.0mg/dl） 无腹泻（或<500ml/d）
+ 斑丘疹<25%体表面积 34-50μmol/L（2.0-3.0mg/dl） 腹泻500~1000ml/d
++ 斑丘疹25-50%体表面积 51-102μmol/L（3.1-6.0mg/dl） 腹泻1000~1500ml/d
+++ 广泛性皮损 103-255μmol/L（6.1-13.2mg/dl） 腹泻>1500ml/d
++++ 剥脱性皮炎伴水疱形成 >255μmol/L（>13.2mg/dl） 严重腹痛伴或不伴肠梗阻、血便
GVHD总分级 皮肤 肝脏 胃肠道 一般状态
Ⅰ +~++ 0 0 正常
Ⅱ +~+++ + + 轻度下降
Ⅲ ++~+++ ++~+++ ++~+++ 中度下降
Ⅳ ++~++++ ++~++++ ++~++++ 重度下降
结果解读 Ⅰ级轻度GVHD，Ⅱ级中度GVHD，Ⅲ~Ⅳ级为重度GVHD，与死亡率显著相关。
相关解释 急性GVHD的Glucksberg分级系统和IBMTR分级系统在评估预后方面仍有各自的优点。
参考来源 王吉耀主编. 《内科学》（八年制）[M]. 人民卫生出版社.2005年.
